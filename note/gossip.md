# 农历与太岁的关系

太岁：木星（昴宿）

农历，与太岁（太岁历）虽然是两个本历法，便现实当中两者却是以一种特殊的方式完美的结合在了一起。而这种的结合便是命理。

农历本身作为指导农事生产和重要历法工具，自然对于中华农业文明有着重要的意义。而同时与中华文明并生的敬鬼神崇先祖有着的祭祀与占卜活动，相对应的对历法工具就是有所需的。故而以天文观察（不以太阳与月亮为独一观测标准）为依据所制定的太岁历便很好的适应了这一需求，随之与农历的结合便很的实现了命理衍算的需求。

# 八字干支
八字，即取“年月日时”四个节点的干支甲子（四支八柱），一共八字。以此为基础而进行命理的推导衍算。

## 历史
八字命理的起源并无去切可考，但可以确定的是随着干支的应用，其雏形便也同时而生。至晚于宋朝，八字命理的内容得到丰富与完善，并被广范使用。
