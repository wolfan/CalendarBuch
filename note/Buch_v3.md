# 关于各术语解释

## 戊日及六戊
戊日是道教中的一个术语，‘戊真不朝’*（相关论文解释是早期规避皇帝形成）*，逢戊日禁诵经、进香，百禁也。其具体禁忌参阅《女青天律》。

- 六戊：即甲子中的六个‘戊’日为明戊。
  - 暗戊为六个月份中日子为暗隐之戊。
